Bug Report ID: TRIAGE-101
Severity: Critical
Description:
The system crashes when trying to index a new repository path that does not exist.  

Steps to Reproduce:
1. Update `appsettings.json` with a non-existent repo path.
2. Run the application.
3. Observe crash on startup.

Expected:
The application should validate the path and show a friendly error message, instead of crashing.
